# Do Tick

Do Tick is a simpke to-do list application that allows you to search for items, automatically checks and ticks completed categories and backs up your data on device persistently 

# Technologies used
- Xcode
- Swift
- CoreData
- Realm
- PLIST
- Storyboards and progmattic UI
- UITableView
- Custom UITableViewCell and XIB file
- MVC Design Pattern
- UIKit
- AutoLayout
- Segues and Navigation
- Cocoapods for Realm
